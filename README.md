# nanp

Connect between the sponsors and the audience

Getting Started
---------------

### Install tools

- yarn
    - `npm install -g yarn@latest`
- react-native-cli
    - `npm install -g react-native-cli`

1. Clone this repo to your local machine
2. Run `yarn`
3. Run `react-native run-ios` / `react-native run-android`
